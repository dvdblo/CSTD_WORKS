# Breakout: the game

## How to Run the Game

Go to the "html" folder and open the "index.html" file in your preferred web browser.

## Game Controls

**A / Left Arrow:** Moves the paddle left.

**D / Right Arrow:** Moves the paddle right.

**Spacebar:** Serves the ball.

**R:** Restarts the game.

## Game Objective

Destroy all the bricks on the screen.

## Game Rules

1. When the ball hits the paddle, it will bounce at an angle determined by the ball's position relative to the paddle.

2. The ball bounces off the side and top walls.

3. Bricks are destroyed when hit by the ball.

4. Bricks may require more than one hit to be destroyed. This is indicated by the color: Green = 1 hit; Yellow = 2 hits; Orange = 3 hits; Red = 4 hits.

5. For every hit to a brick, you recive one point. You can see your score in the center of the screen at any moment.

6. You have three lives in total.

7. If the ball touches the bottom edge, you lose a life.

8. If you reach 0 lives, you lose.

9. You can restart your game at any time by pressing the "r" key.

10. Have fun!

## Aditional gameplay

It was implemented the generation of bricks of different colors in equal positions, thus creating platforms with greater resistance. Resistance increases from lowest to highest, from bottom to top.
